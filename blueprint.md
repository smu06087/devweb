# Lotto Number Generator & Animal Face AI Project Blueprint

## Project Overview
A modern, interactive web-based portal featuring a Lotto number generator, an AI-powered "Animal Face Test" via image upload, and a business partnership inquiry system. Optimized for Google AdSense and SEO.

## Project Details
- **Framework-less Development:** Built using HTML, CSS, and JavaScript with modern standards (Baseline).
- **Web Components:** Custom `<lotto-ball>` and `<ai-result>` elements for modularity.
- **Modern CSS:** Extensive use of OKLCH colors, Container Queries, and Glassmorphism.
- **AI Integration:** Teachable Machine Image Model for real-time animal face classification from uploaded photos.
- **Formspree Integration:** For professional partnership inquiries.
- **AdSense & SEO Optimization:** Includes meta tags, Sitemap, Robots.txt, Privacy Policy, Terms, and About Us pages for high-quality standards.

## Design Specifications
- **Typography:** Pretendard (Modern Korean Font).
- **Color Palette:** Vibrant OKLCH colors with consistent dark/premium aesthetics.
- **Interactivity:** Smooth scroll, staggered animations, and photo-based AI feedback.
- **Navigation:** Transparent navigation and footer links to policy pages.
